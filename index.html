Cours suivi

Concepts clés

Partie 1 - Découvrez le web et ses origines

Web

Le World Wide Web, est un système de pages web publiques interconnectées, accessibles via Internet. Il est l'une des nombreuses applications construites sur Internet.

Internet

Internet est l'ensemble de réseaux mondiaux interconnectés qui permet à des ordinateurs et à des serveurs de communiquer efficacement au moyen d'un protocole de communication commun.

Cloud

Le terme « cloud » désigne les serveurs accessibles sur Internet, ainsi que les logiciels et bases de données qui fonctionnent sur ces serveurs.

W3C

Le W3C est un sigle utilisé pour définir le World Wide Web Consortium qui est une organisation non lucrative permettant définir des standards pour les technologies liées aux web.

Partie 2 - Distinguez les langages du Web

Langages client

Un langage client est un langage de programmation exécuté sur l'ordinateur du client, le code est reçu puis exécuté sur l'ordinateur de la personne qui exécute.

Navigateurs

Un navigateur est un logiciel qui donne accès à Internet et permet à l’utilisateur de consulter des pages web.

Langages serveur

Un langage de script serveur est un type de langage de programmation qui est exécuté côté serveur au lieu de côté client.

Frameworks

Un framework est un ensemble de composants logiciels réutilisables qui permettent de développer de nouvelles applications plus efficacement.

Bases de données

Une base de données est un ensemble d'informations qui est organisé de manière à être facilement accessible, géré et mis à jour.

Site responsive

Le site responsive consiste à rendre un site web accessible et adaptable à tous les devices : tablettes, smartphones, etc.

Application mobile

Une application mobile c’est un programme téléchargeable sur smartphone ou tablette qui comporte un fichier qui est installé puis exécuté par le système d’exploitation de votre mobile.

Partie 3 - Découvrez le fonctionnement du réseau

Serveurs

Un serveur est un ordinateur ou un système qui met des ressources, des données, des services ou des logiciels à la disposition d'autres ordinateurs, qualifiés de « clients », sur un réseau.

Adresse IP

Une adresse IP (Internet Protocol) est le numéro d’identification unique attribué à chaque appareil connecté à Internet.

Nom d’hôte et DNS

DNS signifie 'Domaine Name System' en anglais, en français on peut dire système de noms de domaine. Il permet de traduire le nom d'hôte en adresse IP.

Protocoles

Un protocole est un ensemble normalisé de règles régissant le formatage et le traitement des données. Ils permettent aux ordinateurs de communiquer entre eux.

Partie 4 - Identifiez les métiers du web

Développeurs

Le développeur est un technicien en charge de la réalisation d’un système ou d’une application.

Product Managers et chefs de projets

Le Product Manager est le gardien de la vision produit et du respect de cette dernière, à la différence, le Chef de projet est l’architecte de l’exécution de la vision produit défini par le PM.‍

Designers

Le designer définit la forme et le comportement des produits et des services numériques.

Testeurs

Le testeur assure un travail de préparation et d'analyse. Il établit une stratégie, planifie les opérations, élabore les outils de tests, exécute les tests, analyse les résultats et rédige des rapports.

Techniciens informatique

Un technicien informatique, est un professionnel spécialisé dans la gestion, la maintenance et la résolution des problèmes liés aux systèmes informatiques, matériels et logiciels.

Langages de développement web

HTML

Front-End

Squelette du site Web. Il définit la structure de base du contenu, comme les titres, les paragraphes, les images, etc.

CSS

Front-End

Le CSS correspond à un langage informatique permettant de mettre en forme des pages web (HTML ou XML).

JavaScript

Front-End

JavaScript permet aux développeurs de créer des fonctionnalités interactives pour les utilisateurs, d'animer des éléments sur une page web et de manipuler le contenu HTML et CSS.

JavaScript

Back-End

Javascript est responsable des fonctionnalités, du « back-office » ou de l'admin d'un projet de développement web.

PHP

Back-End

Le PHP (acronyme de Hypertext Preprocessor) est un langage informatique open source permettant de créer des pages web dynamiques.

Java

Back-End

Java est un langage de programmation multiplateforme orienté objet qui s'exécute sur des milliards d'appareils dans le monde entier.

Python

Back-End

Python est un langage de programmation largement utilisé dans les applications Web, le développement de logiciels, la science des données et le machine learning (ML).

C#

Back-End

C# est un langage de programmation orienté objet multiplateforme moderne, novateur et open-source.

SQL

Bases de données

Le langage SQL est un langage de requête populaire fréquemment utilisé dans tous les types d'applications.

Boîte à outils du développeur

  • IDE comme VS Code : Un environnement de développement intégré (IDE) est une application logicielle qui aide les programmeurs à développer efficacement le code logiciel.
  • Github : GitHub est une plateforme basée sur le cloud où vous pouvez stocker, partager et travailler avec d'autres pour écrire du code.
  • Librairies / bibliothèques : Une librairie est un ensemble de codes pré-écrits et compilés, regroupés dans des modules ou des packages, qui offrent une fonctionnalité spécifique.
  • Frameworks : Un cadre de travail déja calibré et prêt à l’emploi qui peut être utilisé facilement selon le projet sur lequel il travaille. Au cœur de ce cadre de travail se trouvent plusieurs langages de programmation associés comme HTML, Python, CSS, Ruby, PHP...
